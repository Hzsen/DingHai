# AI硬件扩散链交易框架

## 0. 核心结论

这套框架的本质不是炒“AI”两个字，而是寻找：

> **AI资本开支落到真实硬件需求之后，哪一层产业链还没有被充分定价。**

当前可以把 AI 硬件扩散链分成七层：

1. **光通信 / 光模块 / 交换机**
2. **PCB / CCL / 铜箔 / 玻纤布**
3. **MLCC / 电感 / 被动元件 / 电源材料**
4. **半导体设备**
5. **半导体材料 / 电子特气 / 晶圆制造耗材**
6. **SiC / GaN / 功率半导体 / 数据中心电源效率**
7. **小金属 / 资源品 / 供给扰动**

交易优先级：

> **先看 AI服务器直接用量，再看晶圆制造耗材，再看电力效率扩散，最后看资源价格传导。**

# 2. A档主线：PCB / CCL / 铜箔 / 玻纤布

## 2.1 PCB为什么是 AI服务器最直接的扩散链？

AI服务器和普通服务器不同，主要变化在于：

- GPU数量更多；
- 信号传输速度更高；
- 服务器主板层数更高；
- 交换机、背板、加速卡、网卡需求增加；
- 对低损耗、高可靠、高散热材料要求提高。

所以 AI服务器扩散链不是只看 GPU，而是要看：

> **GPU → 服务器 → 交换机 → 高速互联 → PCB / CCL / 铜箔 / 玻纤布。**

------

## 2.3 PCB交易重点

### 最强方向

1. **AI服务器高多层 PCB**
2. **高速低损耗 CCL**
3. **HVLP铜箔**
4. **低损耗玻纤布**
5. **服务器 / 交换机背板**

### 关键判断

PCB不是看“有没有 PCB 业务”，而是看：

- 是否做高多层板；
- 是否进入 AI服务器 / 交换机供应链；
- 是否有高速低损耗材料能力；
- 是否有 Nvidia / AMD / 云厂商 / 交换机厂间接订单；
- 毛利率是否上行；
- ASP 是否提升；
- 产能利用率是否提升。

# 3. B档扩散：MLCC / 被动元件 / 电源材料

## 3.1 MLCC的定位

MLCC 是多层陶瓷电容，主要用于：

- 滤波；
- 去耦；
- 储能；
- 稳压；
- 高频信号处理；
- 电源管理。

AI服务器、机器人、汽车电子、工业控制都会提高 MLCC 用量和规格要求。

但是 MLCC 和 PCB 不一样：

> **PCB 是 AI服务器一阶受益，MLCC 是电子系统复杂度提升后的二阶受益。**

------

## 3.2 MLCC产业链拆分

| 子方向                 | 具体材料 / 设备               | AI相关性 | 交易属性                         | 核心验证                       | 风险                 |
| ---------------------- | ----------------------------- | -------- | -------------------------------- | ------------------------------ | -------------------- |
| 高端 MLCC              | 高容、高压、车规、工业级 MLCC | 中强     | AI服务器 / 汽车 / 机器人共同受益 | 高端品占比、价格、稼动率       | 普通消费电子库存周期 |
| 钛酸钡 BaTiO3          | MLCC核心介电陶瓷粉体          | 中强     | 材料壁垒高                       | 粒径、纯度、均匀性、客户认证   | 国内高端粉体突破慢   |
| 镍粉 / 镍浆            | BME MLCC内部电极              | 中强     | 成本 + 技术双逻辑                | 纳米镍粉、浆料分散、烧结稳定性 | 镍价波动             |
| 稀土添加剂             | Dy、Y、Mn、Mg 等              | 中等     | 改善寿命、温度稳定性             | 高端介质配方                   | 主营占比小           |

| 烧结炉                 | 高温共烧设备                  | 中等     | 高端制造设备                     | 气氛控制、温度一致性           | 下游扩产周期影响     |

------

## 3.3 MLCC交易重点

MLCC不是单纯看“电容涨价”，而是看：

1. 是否从消费电子 MLCC 走向高端 MLCC；
2. 是否有车规、工业、服务器客户；
3. 是否能做高容、高压、小型化产品；
4. 钛酸钡粉体、镍粉、浆料是否国产替代；
5. 毛利率是否从库存周期底部修复；
6. 下游是否从手机周期切换到汽车、服务器、机器人周期。

# 4. C档核心：半导体设备

## 4.1 半导体设备的本质

半导体设备的交易逻辑不是“材料涨价”，而是：

> **晶圆厂 Capex → 设备订单 → 交付 → 收入确认 → 毛利率验证。**

设备股看四件事：

1. 晶圆厂是否扩产；
2. 国产替代是否加速；
3. 公司是否拿到订单；
4. 收入确认是否跟上。

## 5.3 六氟化钨 WF6 的单独框架

### WF6是什么？

六氟化钨，英文 Tungsten Hexafluoride，化学式 WF6。

主要用于半导体制造中的：

- 钨薄膜沉积；
- 钨塞；
- 接触孔填充；
- 金属互联；
- 部分 TSV / 先进封装相关工艺；
- CVD / ALD 工艺。

### WF6的交易逻辑

WF6不是大宗材料，而是：

> **小众、高纯、高壁垒、认证周期长的晶圆制造耗材。**

它的催化因素包括：

1. 先进制程增加钨沉积需求；
2. 3D NAND / DRAM / HBM 制造需求提升；
3. 国内晶圆厂扩产；
4. 进口替代加速；
5. 海外供应扰动；
6. 高纯钨粉价格变化；
7. 国内厂商通过晶圆厂认证；
8. 电子特气收入占比提升。

### WF6要警惕的问题

| 风险                         | 解释                               |
| ---------------------------- | ---------------------------------- |
| 只有钨资源，没有半导体级 WF6 | 那是资源股，不是半导体材料股       |
| WF6收入占比太低              | 题材弹性可能大于业绩弹性           |
| 认证周期太长                 | 逻辑很硬，但兑现很慢               |
| 纯度不够                     | 无法进入先进晶圆厂供应链           |
| 安全生产风险                 | 电子特气属于危化品，高壁垒也高监管 |

# 6. E档扩散：SiC / GaN / 功率半导体

## 6.1 SiC的定位

SiC，碳化硅，是宽禁带功率半导体材料。

它不是 AI芯片制造的主线材料，而是：

> **AI数据中心电力侧、能效侧、功率转换侧的二阶受益方向。**

传导路径：

> AI数据中心扩张
> → 电力消耗上升
> → 电源效率要求提高
> → UPS / HVDC / PDU / 电源模块升级
> → SiC / GaN功率器件需求提升

------

## 6.2 SiC产业链拆分

| 环节         | 具体内容                         | AI相关性 | 交易属性        | 核心验证                   | 风险             |
| ------------ | -------------------------------- | -------- | --------------- | -------------------------- | ---------------- |
| 高纯 SiC粉体 | 长晶原料                         | 中等     | 上游材料        | 纯度、稳定供应             | 单独空间有限     |
| SiC晶体生长  | PVT长晶炉                        | 中强     | 设备 + 工艺壁垒 | 晶体尺寸、缺陷密度         | 良率不确定       |
| SiC衬底      | 6英寸 / 8英寸导电型、半绝缘型    | 中强     | 核心瓶颈        | 良率、价格、客户认证       | 衬底过剩、价格战 |
| SiC外延片    | 外延层生长                       | 中强     | 高壁垒          | 厚度均匀性、缺陷控制       | 客户认证慢       |
| SiC器件      | SBD、MOSFET                      | 中强     | 产业链中游      | 导通电阻、可靠性、车规认证 | 海外龙头压制     |
| SiC模块      | 功率模块                         | 中强     | 下游应用        | 车企 / 逆变器 / 电源客户   | 封装可靠性       |
| 设备耗材     | 石墨件、坩埚、涂层件、切磨抛设备 | 中强     | 隐形耗材        | 产能扩张与更换周期         | 周期波动         |

------

## 6.3 SiC应用场景

| 应用        | SiC作用                            | AI相关性 |
| ----------- | ---------------------------------- | -------- |
| 新能源车    | 主驱逆变器、OBC、DC-DC             | 间接     |
| 光伏 / 储能 | 逆变器效率提升                     | 中等     |
| 数据中心    | 电源模块、UPS、HVDC、PDU，提高能效 | 中强     |
| 工业电源    | 高频高压功率转换                   | 中等     |
| 机器人      | 电机驱动、电源管理                 | 中等     |
| 充电桩      | 高压快充                           | 间接     |

------

## 6.4 SiC交易重点

SiC不能简单当成 AI芯片材料炒。

正确归类是：

> **AI电力链 + 新能源车 + 光伏储能 + 工业电源的交叉方向。**

重点看：

1. 6英寸到8英寸切换；
2. 衬底价格是否继续下跌；
3. 外延良率；
4. 器件客户认证；
5. 车规订单；
6. 数据中心电源模块应用；
7. 是否存在产能过剩。

------

# 7. F档：小金属 / 资源品 / 供给扰动

## 7.1 小金属为什么会被 AI 扩散带动？

AI硬件链拉长以后，会传导到：

- 服务器；
- 光模块；
- 半导体；
- PCB；
- 电源；
- 散热；
- 储能；
- 机器人；
- 军工通信。

这些方向会间接拉动某些金属材料需求。

------

## 7.2 小金属细分

| 金属 | 主要用途                      | AI相关性 | 交易逻辑                 | 风险                 |
| ---- | ----------------------------- | -------- | ------------------------ | -------------------- |
| 铜   | 电力、PCB、数据中心、通信     | 强       | AI电力 + PCB + 电网      | 大宗波动             |
| 钨   | WF6、硬质合金、军工、设备材料 | 中强     | 半导体特气 + 高端制造    | 半导体级占比不一定高 |
| 镓   | GaN、化合物半导体             | 中强     | 功率器件、射频、出口管制 | 需求体量有限         |
| 锗   | 光纤、红外、半导体            | 中强     | 光通信、军工、出口管制   | 资源股波动           |
| 铟   | ITO、显示、半导体             | 中等     | 显示 / 电子材料          | AI相关性较弱         |
| 锑   | 阻燃剂、合金、光伏玻璃        | 中等     | 供给扰动、出口管制       | 容易脱离 AI主线      |
| 锡   | 焊料、电子连接                | 中等     | 电子制造恢复             | 周期性               |
| 镍   | MLCC电极、电池、不锈钢        | 中等     | MLCC + 电池              | 价格波动大           |
| 银   | 导电浆料、光伏、电子          | 中等     | 光伏 + 电子              | 贵金属属性干扰       |

------

## 7.3 小金属交易重点

小金属必须区分两种逻辑：

### 资源逻辑

看：

- 金属价格；
- 库存；
- 供给扰动；
- 出口管制；
- 资源储量；
- 冶炼产能。

### 半导体材料逻辑

看：

- 是否能做到电子级；
- 是否进入晶圆厂；
- 是否有高纯材料收入；
- 是否有客户认证；
- 是否有毛利率提升。

最容易出错的是：

> **钨价上涨 ≠ WF6半导体材料逻辑兑现。**

如果公司只是有钨矿，没有半导体级六氟化钨认证，那么它是资源股，不是半导体材料股。

------

# 8. 交易优先级排序

## 8.1 第一优先：AI服务器直接链

| 方向           | 原因                          |
| -------------- | ----------------------------- |
| 高多层 PCB     | AI服务器 / 交换机直接用量提升 |
| 高速低损耗 CCL | PCB性能瓶颈                   |
| HVLP铜箔       | 高速信号材料瓶颈              |
| 电子玻纤布     | 低损耗、低热膨胀材料          |
| 先进封装材料   | HBM / Chiplet直接相关         |

特点：

- 离 AI服务器订单最近；
- 容易被业绩验证；
- 容易走成主线；
- 风险是估值拥挤和高位补跌。

------

## 8.2 第二优先：半导体国产替代 + 晶圆制造耗材

| 方向               | 原因                       |
| ------------------ | -------------------------- |
| WF6 六氟化钨       | 小众高壁垒，先进制造不可缺 |
| NF3 / ClF3         | 晶圆厂高消耗材料           |
| CMP抛光液 / 抛光垫 | 先进制程和平坦化必要耗材   |
| 靶材               | PVD沉积核心材料            |
| 光刻胶 / 配套材料  | 国产替代想象力强           |
| 半导体设备零部件   | 国产替代 + 设备扩产        |

特点：

- 逻辑硬；
- 认证周期长；
- 弹性来自国产替代、供给扰动和客户导入；
- 风险是收入占比低、业绩兑现慢。

------

## 8.3 第三优先：AI电力效率链

| 方向             | 原因                         |
| ---------------- | ---------------------------- |
| SiC衬底          | 功率半导体核心材料           |
| SiC外延 / 器件   | 新能源车、储能、数据中心电源 |
| GaN功率器件      | 高频电源、通信、快充         |
| 数据中心电源模块 | AI电力链直接受益             |
| UPS / HVDC / PDU | 数据中心供电架构升级         |

特点：

- 和 AI有关，但不是 AI芯片正线；
- 更像 AI电力链二阶扩散；
- 风险是传导链长、价格战、产能过剩。

------

## 8.4 第四优先：被动元件与电子材料扩散

| 方向                | 原因                         |
| ------------------- | ---------------------------- |
| 高端 MLCC           | AI服务器、汽车、机器人都需要 |
| 钛酸钡粉体          | MLCC核心介电材料             |
| 纳米镍粉 / 镍浆     | MLCC内部电极材料             |
| 高端电感 / 磁性材料 | 电源管理需求提升             |

特点：

- 适合补涨；
- 需要结合消费电子库存周期；
- 业绩验证看稼动率和毛利率；
- 不适合无脑追 AI概念。

------

# 10. 不同阶段的交易策略

## 10.1 主线启动期

市场特征：

- 光模块、算力、服务器先涨；
- 指数风险偏好修复；
- AI Capex 预期升温。

优先方向：

1. 光模块；
2. 交换机；
3. 高多层 PCB；
4. CCL；
5. 先进封装。

不宜优先追：

- 小金属；
- MLCC上游；
- SiC远端扩散；
- 纯题材半导体材料。

------

## 10.2 主线扩散期

市场特征：

- 龙头开始高位震荡；
- 资金寻找低位补涨；
- 产业链开始向材料、设备、资源扩散。

优先方向：

1. PCB上游；
2. HVLP铜箔；
3. 电子玻纤布；
4. 半导体设备材料；
5. 电子特气；
6. CMP材料；
7. MLCC高端化。

------

## 10.3 业绩验证期

市场特征：

- 市场不再只相信概念；
- 开始看订单、收入、利润；
- 蹭概念公司退潮。

重点看：

1. 毛利率；
2. ASP；
3. 产能利用率；
4. 合同负债；
5. 存货；
6. 客户结构；
7. 收入占比；
8. 净利润弹性。

------

## 10.4 退潮期

市场特征：

- 题材高位补跌；
- 扩散方向冲高回落；
- 业绩不兑现的公司杀估值。

优先规避：

1. 收入占比极低的概念股；
2. 只有互动平台回复的公司；
3. 价格已经透支两年以上业绩的公司；
4. 金属价格回落但股价仍在高位的资源股；
5. 产能过剩的 SiC衬底公司。

------

# 11. 核心混淆点

## 11.1 PCB和半导体材料不是一回事

| 项目     | PCB                       | 半导体材料           |
| -------- | ------------------------- | -------------------- |
| 用途     | 电子系统连接              | 晶圆制造             |
| 订单验证 | 较快                      | 较慢                 |
| 客户     | 服务器厂、交换机厂、EMS厂 | 晶圆厂、封测厂       |
| 关键指标 | 层数、低损耗、ASP、订单   | 纯度、认证、制程导入 |
| 交易节奏 | 更快                      | 更慢                 |

------

## 11.2 SiC不是 AI芯片正线

SiC 正确归类：

> **功率半导体 / 电力电子 / 数据中心电源效率链。**

错误归类：

> **AI芯片制造材料。**

------

## 11.3 钨资源不等于 WF6

| 类型                 | 逻辑                 |
| -------------------- | -------------------- |
| 钨矿 / 钨粉          | 资源价格逻辑         |
| 高纯钨粉             | 半导体材料上游       |
| WF6六氟化钨          | 晶圆制造电子特气     |
| 通过晶圆厂认证的 WF6 | 真正的半导体材料逻辑 |

------

## 11.4 MLCC不是第一波 AI主线

MLCC 更适合：

- AI硬件扩散；
- 汽车电子复苏；
- 机器人放量；
- 消费电子补库存；
- 高端被动元件国产替代。

不适合：

- 把普通消费电子 MLCC 强行当 AI服务器核心标的。

------

# 12. 最终总结

## 12.1 一句话总框架

> **PCB是 AI服务器的“血管”，MLCC是电子系统的“毛细血管”，WF6是晶圆制造的“小而硬耗材”，SiC是 AI电力侧的“效率器件”。**

------

## 12.2 交易优先级压缩版

| 优先级   | 方向                              | 交易逻辑                            |
| -------- | --------------------------------- | ----------------------------------- |
| 第一优先 | PCB / CCL / HVLP铜箔 / 玻纤布     | AI服务器直接用量，容易订单验证      |
| 第二优先 | 电子特气 / WF6 / NF3 / CMP / 靶材 | 晶圆制造耗材，国产替代 + 认证突破   |
| 第三优先 | 半导体设备零部件                  | 设备国产化、晶圆厂扩产、耗材替代    |
| 第四优先 | MLCC / 钛酸钡 / 镍粉              | 高端电子系统复杂度提升后的补涨      |
| 第五优先 | SiC / GaN / 功率模块              | AI电力效率链二阶扩散                |
| 第六优先 | 小金属 / 资源品                   | 供给扰动 + 价格弹性，但要防业绩证伪 |

------

## 12.3 最终交易原则

1. **先看直接订单，再看远端扩散。**
2. **先看主营占比，再看题材想象。**
3. **先看客户认证，再看新闻催化。**
4. **先看毛利率和 ASP，再看股价涨幅。**
5. **先看产业链位置，再看市场情绪。**
6. **纯概念可以炒一波，但不能当长期主线。**
7. **真正能走二波三波的，必须有业绩验证。**
