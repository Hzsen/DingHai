# Resume Bullets

- Built a LangGraph-based financial research agent that routes natural-language questions across SQLite tools, BM25 keyword search, local vector retrieval, factor documentation, and backtest-result tables.
- Designed safe parameterized SQL tools for structured financial metrics, including regime performance, factor comparisons, strategy drawdown, turnover, anomaly logs, and Sharpe-based rankings.
- Implemented hybrid retrieval and deterministic answer verification so numerical claims are grounded in SQL results and causal explanations cite retrieved research notes.
- Added reproducible CLI commands, sample financial datasets, unit tests, and evaluation scripts for routing accuracy, retrieval quality, and answer grounding.
